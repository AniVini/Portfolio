I built the entire system—from data preprocessing and model training to frontend integration and deployment—ensuring it’s both accurate and user-friendly. Ideal for use in cybersecurity tools, this project showcases my skills in Python, ML algorithms, Flask, and web development.

Music Genre Classifier

This project focuses on building a machine learning model capable of identifying the genre of a music track—such as Pop, Classical, or Rock—based on audio characteristics. By training the model on a labeled dataset, the system learns to recognize patterns in pitch, rhythm, tempo, and frequency that are unique to each genre. I used audio processing techniques to extract relevant features, which were then fed into supervised learning algorithms for genre classification. The result is an intelligent tool that can analyze an audio input and accurately predict its genre. This project highlights my understanding of audio signal processing, feature engineering, and machine learning implementation in multimedia contexts.
